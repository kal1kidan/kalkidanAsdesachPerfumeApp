# 🌸 kal Perfumes

A luxury perfume ecommerce mobile application built with Flutter, designed with cinematic UI/UX, premium animations, and elegant fragrance-inspired aesthetics.

The app delivers a modern high-end shopping experience inspired by luxury perfume brands such as Dior, Chanel, Tom Ford, and Maison Francis Kurkdjian.

---

# ✨ Features

* Luxury perfume shopping experience
* Cinematic onboarding screens
* Elegant product detail pages
* Smooth animations and transitions
* Responsive modern Flutter UI
* Wishlist and cart functionality
* Premium checkout flow
* Clean architecture and reusable components
* Material 3 design system
* Light and dark theme support

---

# 🎨 Design Style

The application follows a luxury editorial design language featuring:

* Deep burgundy, beige, and gold color palette

# 📋 Prerequisites

Before running the project, make sure you have:

* Flutter SDK
* Dart SDK
* Android Studio or VS Code
* Android SDK

Check Flutter installation:

```bash
flutter doctor
```

---

# 🛠️ Installation

Clone the repository:

```bash
git clone https://github.com/kal1kidan/kalkidanAsdesachPerfumeApp.git
```

Navigate into the project:

```bash
cd kalkidanAsdesachPerfumeApp
```

Install dependencies:

```bash
flutter pub get
```

Run the application on Android:

```bash
flutter run
```
